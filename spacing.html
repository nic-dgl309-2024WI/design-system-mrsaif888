Spacing

Maring and Padding

The class to represent margin is m
The class to represent padding is p

Margin and Padding Sides

  • A class that represent margin-top or padding-top is t
  • A class that represent margin-bottom or padding-bottom is b
  • A class that represent margin-left or padding-left l
  • A class that represent margin-right or padding-right is r
  • A class that represent both margin left and right or padding left and right is x
  • A class that represent both margin top and bottom or padding top and bottom is y

Margin and Padding Sizes

  • classes that reset margin or padding to 0 is 0
  • classes that reset margin or padding to 5 is 1
  • classes that set margin or padding to 1 is 2
  • classes that set margin or padding to 1.5 is 3
  • classes that set margin or padding to 2 is 4
  • classes that set margin or padding to 2.5 is 5
  • classes that set margin or padding to auto is auto
